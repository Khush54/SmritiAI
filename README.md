# 🧠 Smriti AI – Frontend

🚧 **Status: Under Development**

Smriti AI is an AI-powered platform focused on early-stage dementia detection.  
This repository contains the frontend of the application, currently under active development.

---

## 🚀 Current Progress

- UI design and layout setup
- Landing page development
- Basic dashboard structure
- Responsive design (in progress)

---

## 🔜 Upcoming Features

- Multi-modal test interface (Voice, Drawing, Memory, Behaviour)
- AI-based risk reports and analytics
- Doctor & caregiver dashboard
- Authentication system (Login/Signup)
- API integration

---

## 🛠️ Tech Stack

- React.js  
- Tailwind CSS  
- Chart.js  

---

## ⚙️ Setup Instructions

git clone https://github.com/Khush54/SmritiAI.git  
cd SmritiAI  
npm install  
npm run dev
